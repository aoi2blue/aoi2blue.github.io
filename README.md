日本時間3/1AMより発生しています。<br>
NSOアプリが更新されたことで、salmdroidが連携/取得時に利用するログイン用サーバの対応が必要となっています。<br>
対応が完了するまでお待ち下さい。<br>

## リザルト取得時にSummaryで失敗する
お手数をおかけし申し訳ありません。<br>
ver2.0.0から環境によって発生しています。下記をお試しください(連携解除しても保存してあるリザルトは消えません)。<br>
1. settingsタブを開く
2. 「アカウント連携を解除」を
3. 確認のダイアログが表示されるのでOKをタップ
4. アプリを起動する
5. アカウント連携を行う
